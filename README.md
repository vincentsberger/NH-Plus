# NH-Plus #

## Was ist NH-Plus? ##
NH-Plus ist eine Verwaltungssoftware für medizinische Einrichtungen. Es können Pfleger, Patienten und Behandlungen erstellt und verwaltet werden. Mehr Features folgen noch.
